##  Overview

This repository contains the **Phishing Email Detection & Awareness Report**
prepared as part of the **Future Interns Cyber Security Internship — Task 2**.

The report analyzes real-world phishing email patterns, identifies common
attack indicators, classifies email risk levels, and provides practical
prevention guidelines for employees and businesses.

---

##  Objective

- Analyze phishing email samples
- Identify common phishing indicators
- Classify emails as Safe / Suspicious / Phishing
- Create an awareness document businesses can actually use

##  Key Phishing Indicators Found

- Fake or spoofed sender domains
- Urgency and fear-based language
- Suspicious or mismatched links
- Generic greetings (no real name)
- Requests for sensitive information
- Missing personalisation
